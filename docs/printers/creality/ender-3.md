# Ender-3

Comes in a few varieties now, with single Z axis stepper.

The standard Ender-3 has a 0.4 nozzle, a print surface 235x235 with a Z of 250. The nominal print area is 220x220, but one can get prints to 230x230.

Later versions have a few improvements to the Extruder, including a brass insert in the Extruder lever, because putting a few dozen spools through the printer if it's in standard build will wear a slot in the lever and it seizes up.
