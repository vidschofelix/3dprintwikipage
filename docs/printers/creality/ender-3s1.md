# Creality Ender-3 S1
## General Info
- Manufacturer [Creality](https://www.creality.com/goods-detail/creality-ender-3-s1-pro-fdm-3d-printer)

## Specifications

::: details Show specification table

| Property | Value |
|---|---|
| Model | Ender-3 S1 |
| Machine Weight | 8.6 kg |
| Gross Weight | 10.7 kg |
| Machine size | 487 * 453 * 622 mm |
| Packing size | 540 * 510 * 260 mm |
| LCD Display | 4.3" color touch screen |
| Supported Material | PLA, ABS, TPU, PETG |
| Material Diameter | 1.75 mm |
| Frame | aluminum profiles |

| Property | Value |
|---|---|
| Print Size | 220 * 220 * 270 mm |
| Printing Speed | max 150 mm/s |
| Nozzle Diameter | 0.4 mm |
| Layer thickness | 0.05-0.35 mm |
| Printing Accuracy | 0.01 mm |
