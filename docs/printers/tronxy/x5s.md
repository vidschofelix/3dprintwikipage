# Tronxy X5S
## General Info
### Specification

| Property | Value |
|---|---|
| Model | X5S |
| Machine Weight | 7.6kg |
| Gross Weight | 13kg |
| Machine size | 658 * 580 * 639mm |
| Packing size | 580 * 380 * 190mm |
| Consumable Material | ABS/PLA/HIPS/etc. |
| Material Diameter | 1.75mm |
| Framework | Aluminium profile |

### Technical Parameters

| Property | Value |
|---|---|
| Print Size | 330 * 330 * 400mm |
| Printing Speed | 20-150mm/s |
| Nozzle Diameter | 0.4mm |
| Layer thickness | 0.1-0.4mm |
| max. Extrusion Head Temperature | 270°C |
