# Cura
## Cura 2.1
### Configuraton files(.json)
The configuration of the printer is defined by json files. Copy these into the folder C:\Program Files\Cura 2.1\resources\machines (Windows) or ~/.cura/machines (Mac).
- **Anet A8 with autolevel and nozzle initialization at the corner** [aneta8withinit.zip](/images/reprap/anet/a8/slicer/aneta8withinit.zip)
- Anet A8 with autolevel (NOT tested) [aneta8.zip](/images/reprap/anet/a8/slicer/aneta8.zip)
- Anet A8 with autolevel and reduce bed size to avoid colision between official Anet sensor and heatbed connector. A nozzle initializartion in the corner is added. [aneta8reducedwithinit.zip](/images/reprap/anet/a8/slicer/aneta8reducedwithinit.zip)
