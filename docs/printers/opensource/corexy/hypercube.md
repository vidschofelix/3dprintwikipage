# Hypercube
![](/images/printers/opensource/hypercube.jpg) 

The [HyperCube](http://www.thingiverse.com/thing:1752766) is a design by [Tech2C](http://www.thingiverse.com/Tech2C/) using 20x20 aluminium profiles. The printer is generally very lightweight, and uses CoreXY kinematics.

You can check out the build with a ton of very useful information on [Tech2C](http://www.thingiverse.com/Tech2C/)'s youtube channel: [Build Log Video Playlist](https://www.youtube.com/playlist?list=PLIaArjwViQRVAERWRrYfe9rtiwvvRGCzw)

## BOM

T-Slot 2020 Aluminium Extrusion Profiles for X200 x Y200 x Z155 print bed area

- 4 x 340mm (X)
- 4 x 303mm (Y)
- 4 x 350mm (Z)
- 2 x 285mm (Bed)
- 1 x 135mm (Bed)
