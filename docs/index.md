## Why a wiki?

Discord, subreddits, and forums move on. Answers get lost in the scroll. This wiki catalogues the durable information: what a printer is, what breaks on it, how to fix it, and which mods are worth the time.

## Contributing

Edit any page with the **Edit this page on GitHub** link in the footer, or open a pull request directly at [github.com/vidschofelix/3dprintwikipage](https://github.com/vidschofelix/3dprintwikipage). No account required to read; a GitHub account is all you need to contribute.
